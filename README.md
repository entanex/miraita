## 简介 <img src="https://raw.githubusercontent.com/fenxer/llm-things/3eaaba79ddf48a784304493adfbaa614f410d6e6/images/human-coded.svg" width="90" />

一款跨平台、功能性机器人

## 特性

使用林汐，不再担心：

- 指令碰撞
- 数据丢失
- 帮助不全
- 管理不便

转而担心：

- 地球消失
- 太阳爆炸
- 作者跑路

## 功能

**答案就在身边**，大部分时间您发送 `/help` 即可获得帮助
